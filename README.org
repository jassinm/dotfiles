* Installation

** Requirements

*** OSX

#+begin_src
brew install stow
#+end_src

*** Debian

** Install

#+begin_src
git clone --recurse-submodules -j8 https://github.com/jassinm/dotfiles  ~/.dotfiles
sh bootstrap.sh
#+end_src


** Update
#+begin_src
git pull
git submodule sync --recursive
git submodule update --init --recursive
#+end_src

** Change Shell
#+begin_src
chsh -s /bin/zsh
#+end_src

* Setup

** Tmux

start tmux then run ~Control-b I~ to install the [[tpm][https://github.com/tmux-plugins/tpm]] plugins

** emacs

#+begin_src
cd .emacs.d/bin
./doom install
#+end_src
